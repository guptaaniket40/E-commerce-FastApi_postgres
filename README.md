# 🛒 FastAPI E-commerce API

Simple beginner project using **FastAPI + PostgreSQL**.

## 🚀 Features

* Product CRUD (Create, Read, Update, Delete)
* Cart (Add, View, Increase, Decrease, Remove)
* Order & Dummy Payment

## ⚙️ Setup

```bash
pip install -r requirements.txt
uvicorn app.main:app --reload
```

## 🌐 API Docs

```bash
http://127.0.0.1:8000/docs
```

## 📌 Note

* No authentication
* Dummy payment only
